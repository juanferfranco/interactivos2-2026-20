## Propósito

La fase de investigación es el momento para explorar el territorio técnico, conceptual y estético del proyecto. Durante estas cuatro semanas deberás probar los componentes fundamentales de tu sistema performativo antes de comprometerte con una forma definitiva de obra.

El objetivo es que investigues mediante experimentos: audio, visuales, entrada del performer, entrada del público y concepto general del proyecto.

## Tiempo

Esta fase dura **4 semanas**.

## Entregables

| Entrega | Porcentaje | Tipo |
| --- | ---: | --- |
| Experimento de audio | 5% | Conversación con el profesor |
| Experimento de visuales | 5% | Conversación con el profesor |
| Experimento de entrada del performer | 5% | Conversación con el profesor |
| Experimento de entrada del público | 5% | Conversación con el profesor |
| Concepto del proyecto con texto e imágenes | 5% | Conversación con el profesor |

En cada conversación debes mostrar tu bitácora, el historial de commits y argumentar qué hiciste, por qué lo hiciste y qué aprendiste.

```javascript
setcpm(35) // 140 beat

// ==========================================
// MEZCLADORA INDEPENDIENTE
// ==========================================
const BASS_ON    = 0
const PIANO_ON   = 0
const CEL = 0
const VAR3 = 0
const VAR1  = 0
const VAR2 = 0

// Bass: Sigue las tónicas de la progresión (Fmaj7 - G6 - Em7 - Am9)
bass: note("{<f1 g1 e1 a1>}")
  .sound("supersaw")
  .lpf(slider(1187.4,300,2000)) 
  .room(1)
  .postgain(BASS_ON ? 0.3 : 0)

piano : note("{<[f3,a3,c4,e4] [g3,b3,d4,e4] [e3,g3,b3,d4] [a3,c4,e4,b4]>}")  
  .sound("piano")
  .attack(slider(0.192,0,1)) 
  .lpf(slider(1490,300,2000))
  .room(1.2) 
  .postgain(PIANO_ON ? 0.5 : 0)

// ==========================================
// CÉLULAS Y VARIACIONES MELÓDICAS
// ==========================================
const celula = "<[c5 e5] [b4 g5] [a4 d5] [b4 e5 c5]>"
const var1   = "< [[c5 e5] [b4 g5]] [[c5 e5] [b4 g5]] [[a4 d5] [b4 e5 c5]] [[a4 d5] [b4 e5 c5]]>"
const var2   = celula.transpose(12)
const var3   = "{<[e5 c5] [f5 a4] [g5 d5] [f5 c5 e5]>}"

// ==========================================
// CANALES DE MELODÍA INDEPENDIENTES
// ==========================================

// Célula
$celula: note(celula)
  .sound("sine") 
  .decay(0.4)
  .sustain(0.2)
  .room(1.5) 
  .postgain(CEL ? 0.6 : 0)

// Disminución rítmica
$var1: note(var1)
  .sound("sine") 
  .decay(0.4)
  .sustain(0.2)
  .room(1.5) 
  .postgain(VAR1 ? 0.5 : 0)

// Transpuesta (Octava Alta)
$var2: note(var2)
  .sound("triangle") // sine, triangle
  .decay(0.4)
  .sustain(0.2)
  .room(1.5) 
  .postgain(VAR2 ? 0.2 : 0) 

// Inversión
$var3: note(var3)
  .sound("sine") 
  .attack(slider(0.359,0,1))
  .lpf(slider(993.6,300,2000))
  .postgain(VAR3 ? 0.3 : 0)
```
